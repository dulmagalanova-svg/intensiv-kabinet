Из чего собран каждый агент, чем подключён, какие скиллы — и как собрать себе такой же рой

Наша команда — это рой из Claude Code агентов на одном сервере: общая память (gbrain), общий вход через Telegram, у каждого — своя роль, свой бот и свои инструменты. Принцип роя: build · remember · grow (строить · помнить · расти).

2Система памяти gbrain — общий мозг команды

Это то, что превращает отдельных агентов в команду: общая память + канал координации.

gbrain — общий второй мозг всех агентов. База на сервере: Postgres + pgvector (векторный поиск по смыслу) + FastEmbed (multilingual). Доступ — через MCP-инструменты, не руками.

СерверПортЗачем
gbrain-memory8767запись: решения, error-паттерны, рунбуки, заметки
gbrain-recall8768семантический поиск по всей памяти команды (по смыслу)
gbrain-swarm8766шина координации: агенты передают задачи друг другу (notify/broadcast/escalate)
gbrain-tasks—задачи, статусы, heartbeat агентов (4-й слой модели)
💡 Главное: что записал один агент — видят все. Germiona сохранила error-паттерн → Hormozi и Warren найдут его через recall. Это общий, а не личный слой. Между собой агенты общаются НЕ через Telegram, а через swarm-шину.

4Как собрать себе такой же рой

Один промпт + один репозиторий → рабочая команда за ~90 минут. Пошагово:

1
Склонировать основу: agentos-skills — там архитектура и шаблоны.
2
Прочитать главный документ AGENTIC-SWARM.md (~15 мин) — как устроен рой целиком.
3
Пройти боевой урок lesson-3-agents-with-gbrain (9 шагов, ~90 мин) — создаёшь трёх агентов.
4
Заполнить личности — 4 шаблона CLAUDE.md (кодер / second brain / маркетолог / продажи) под свои роли.
5
Поднять сервисы: общий мозг gbrain (Postgres+pgvector) + Telegram (dashi-plugin) + генератор workspace.
6
Поставить скиллы: Superpowers (обязательно) + прикладные под роли (senior-brainstorm, telegram-bot-builder, yt-research…).
7
Связать через gbrain-swarm — агенты начинают передавать задачи друг другу. Рой готов.
Философия репозитория: «Один промпт. Один репозиторий. Никаких догадок.» Ученик даёт своему агенту ссылку на agentos-skills и промпт «подними мне команду из трёх агентов» — агент читает документацию сам и собирает рой.
