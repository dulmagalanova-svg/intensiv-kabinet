Программы для установки

VS Code — редактор + терминал
Бесплатный редактор, где удобно держать файлы агента и встроенный терминал.
code.visualstudio.com →
Node.js — платформа (ставить первой)
Скачай версию LTS. Без неё не установится Claude Code.
nodejs.org →
Claude Code — «тело» агента
Ставится одной командой в терминале (после Node.js):
npm install -g @anthropic-ai/claude-code
bun — движок плагина
Нужен, чтобы агент-заготовка запустилась. Команда установки:
curl -fsSL https://bun.sh/install | bash
